


# En debut de saison

- Installer WPILib
- Installer les NI game tools
- Sortir la carte SD du roborio et brancher dans PC (avec adaptateur USB)
- Flasher le roborio (avec BalenaEtcher)
- Remettre la carte SD dans roborio et redemarrer
- Lancher roboRio imaging tool
- Configurer le numero dequipe et appuyer Edit Startup Settings
- Brancher le roborio au PC via USB (cable USB A-B)
- Lancer Phoenix Tuner X (Canbus)
- Mettre a jour firmware de TOUS les 13 devices
- redemarrer le robot
